# A2_iOS_Shalev_101482699

A SwiftUI iPhone app that browses a Core Data product catalog. Launches into a per-product browse screen with previous/next navigation, plus a full list view, a search tab, and a form to add new products.

## Features

- Browse view that shows one product at a time with previous and next buttons
- List view with name, description, and price
- Search tab that filters by name or description
- Add a new product
- Delete a product from the detail screen
- Settings tab with a reset button to restore the seeded data

10 products are loaded the first time you launch the app.
